Si vous n'avez pas `vite`, lancez la commande :
`npm install vite`

Afin de télécharger le projet :

```
git clone git@github.com:FalconFabio/GPE.git
```

ou

```
git clone https://github.com/FalconFabio/GPE.git
```

Puis pour lancer le projet

```
cd test2-gpe
```

```
npm install
```

```
npm run dev
```
